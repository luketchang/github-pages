## ğŸ”§ How to Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on **Settings** tab
3. Scroll down to **Pages** section (in the left sidebar under "Code and automation")
4. Under **Source**, select the branch you want to deploy (e.g., `main` or your current branch)
5. Select **/ (root)** as the folder
6. Click **Save**
7. Wait a few minutes for GitHub to build and deploy your site
8. Your site will be live at `https://luketchang.github.io/github-pages/`
